# Prompt Engineering

## 概述

Prompt Engineering 是与大语言模型高效协作的核心技能。本板块聚焦实用的提示词设计方法论，帮助开发者在实际项目中获得更稳定、更可靠的模型输出。

## 内容规划

- Prompt 设计原则
- 少样本学习（Few-shot Learning）
- 思维链（Chain of Thought）
- 结构化输出控制
- 常见陷阱与排错
