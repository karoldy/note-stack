# MCP

## 概述

Model Context Protocol (MCP) 是 Anthropic 提出的开放协议，为 LLM 与外部工具、数据源之间提供标准化的连接方式。通过 MCP Server，LLM 可以安全、可控地访问数据库、API、文件系统等外部资源。
