# AI

## 概述

AI 板块聚焦大语言模型（LLM）及其工程化实践，面向一线开发者，关注可落地的技术方案与工具链。

不同于学术论文解读，这里的每一篇文章都围绕一个实际开发场景展开：如何调用 API、如何设计 Prompt、如何构建 Agent、如何将 LLM 集成到现有系统中。

## 学习路径建议

1. **入门**：从 [LLM](./llm/index.md) 开始，了解基本概念和主流模型
2. **实践**：阅读 [Prompt Engineering](./prompt-engineering/index.md)，掌握与 LLM 高效协作的方法
3. **进阶**：深入 [Agent](./agent/index.md) 和 [MCP](./mcp/index.md)，构建可调用外部工具的智能体
4. **落地**：参考 [AI Engineering](./ai-engineering/index.md)，将 LLM 能力集成到生产系统
