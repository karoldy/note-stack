# AI Engineering

## 概述

AI Engineering 关注将大语言模型能力集成到生产系统的工程实践。与模型研究不同，这里聚焦于可维护、可扩展、可观测的 LLM 应用架构。

## 内容规划

- LLM 应用架构模式
- RAG（检索增强生成）设计
- 评估体系与质量保障
- Token 成本控制策略
- 生产环境部署与监控
