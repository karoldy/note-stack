# 提示词模板

## 概述

本节收录经过实际验证的 Prompt 模板，覆盖日常开发中的高频场景。与 [[提示词工程](../prompt-engineering/index.md) 不同，这里侧重于**可直接复制使用的实战模板**，而非方法论。

## 模板分类

> 📌 模板正在建设中，后续将按场景分类补充。

| 分类 | 说明 |
|------|------|
| 代码生成 | 组件、API、工具函数等生成类 Prompt |
| 代码审查 | Bug 发现、性能分析、安全审计 |
| 重构优化 | 代码简化、性能优化、架构改进 |
| 文档写作 | README、API 文档、变更日志 |
| 架构设计 | 技术方案、系统设计、技术选型 |
| Debug 调试 | 错误分析、日志诊断、异常定位 |

## 使用方式

1. 根据场景找到对应模板
2. 替换模板中的占位符（`{项目名}`、`{文件路径}` 等）
3. 根据实际情况补充上下文
4. 粘贴到 Claude Code / ChatGPT / Cursor 中使用
