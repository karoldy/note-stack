# Express

## 概述

Express 是 Node.js 最老牌、最流行的 Web 框架，也是 NestJS 底层的默认 HTTP 平台适配器。它以中间件机制和极简 API 著称。

## 核心主题（规划中）

- 中间件设计与执行顺序
- 路由组织（Router 拆分）
- 错误处理中间件
- 模板引擎集成（EJS、Pug）
- 安全最佳实践（Helmet、CORS、Rate Limiting）
- 性能优化（compression、caching）
- 从 Express 迁移到 NestJS
