# Node.js

## 概述

Node.js 是 JavaScript 的服务端运行时，基于 Chrome V8 引擎和 libuv 事件循环。它是 NestJS、Express、Fastify 等后端框架的底层基石。

## 核心主题（规划中）

- 事件循环（Event Loop）与 libuv
- Stream 流处理
- Buffer 与二进制数据
- Worker Threads 多线程
- 内存管理与 GC 调优
- 进程管理（Cluster、PM2）
- 错误处理与优雅退出
